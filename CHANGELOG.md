# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Full CI/CD pipeline with GitHub Actions
- Comprehensive error boundaries for graceful error handling
- Sentry error monitoring and performance tracking
- Web Vitals performance monitoring
- Professional logging system
- 17 unit tests with Vitest
- Enterprise-grade documentation (README, Architecture, Contributing, etc.)
- Multi-currency support for investments
- Shared workspaces for collaboration
- AI-powered financial insights

- Improved error handling across all features

### Fixed

- Type consistency for transaction IDs
- Module resolution in test configuration
- localStorage mock issues in tests
